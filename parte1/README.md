# Positivus Landing Page

Reprodução fiel das 4 primeiras seções da landing page Positivus a partir do design no Figma.


## Seções implementadas
- Navbar
- Hero
- Services
- CTA (Let's make things happen)
- Case Studies

## Stack
- React + Vite + TypeScript
- Styled Components
- lucide-react

## Como rodar localmente
```bash
npm install
npm run dev
```
